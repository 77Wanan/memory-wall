記憶牆 (Memory Wall)

一個純前端、零配置的個人記憶管理工具。只需一個 HTML 檔案，就能記錄靈感、學習筆記與日常生活。

功能：
支援短文快速記錄與長文詳細撰寫，可選填標題
預設六大分類（學習、日常、技術、靈感、心情），自動分組顯示
輸入關鍵字即時搜尋，快速找到需要的記憶
所有記憶以列表形式呈現在左側側欄，點擊即看
滑到卡片右上角即可修改或刪除
右上角按鈕一鍵切換白晝與夜間模式
點擊卡片彈出視窗，完整閱讀內容

快速開始：
下載 index.html，雙擊用瀏覽器打開，開始寫第一條記憶。無需安裝、無需伺服器、無需網路。

資料儲存：
所有資料保存在瀏覽器本地，關閉頁面不會遺失。清除瀏覽器快取前請記得備份。

技術：
純 HTML + CSS + JavaScript，無任何第三方依賴，自適應佈局。
